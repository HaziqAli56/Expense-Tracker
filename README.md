# Expense Tracker

A modern Flask-based Expense Tracker web application designed to help users manage their personal finances with ease and precision:

- User Authentication: Secure login and signup system with robust session management to ensure your financial data remains private and safe.

- Transaction Management: Seamlessly add, edit, or delete transactions. Track both income and expenses with support for custom categories and sub-categories.

- Monthly Budget System: Plan and track your financial goals effectively by setting and monitoring monthly budget limits.

- AI Financial Assistant: Integrated AI chatbot powered by the Groq API. It understands natural language (English & Roman Urdu) to assist with expense queries, budgeting advice, and smart financial summaries based on your history.

- Visual Analytics: Interactive charts powered by Chart.js to help you visualize your spending habits and identify financial trends at a glance.

- PDF Report Generation: Generate professional, automatic PDF reports for your monthly summaries, making it easy to store or share your financial status.

- Responsive Design: A clean, mobile-friendly user interface built with Bootstrap 5, ensuring a smooth experience across all devices.

- Form Validation: Integrated Flask-WTF to handle secure and validated form submissions, preventing invalid data entry.

- Database Management: Powered by SQLite with SQLAlchemy ORM for reliable and efficient data storage.

## Technologies Used

This project is built using a modern stack of technologies:

- Backend: Python, Flask, Flask-Login, Flask-SQLAlchemy, Flask-WTF

- Database: PostgreSQL, SQLAlchemy, Psycopg2

- Frontend: Bootstrap 5, HTML5, CSS3, JavaScript

- Charts: Chart.js

- AI Integration: Groq API (Llama 3.3)

- Data Processing: Pandas, NumPy

- Reporting: ReportLab (PDF)

- Utilities: python-dotenv (Environment variables), Requests, Pillow, Pydantic
