# Expense Tracker

A modern Flask-based Expense Tracker web application with:

- User Authentication
- Add/Edit/Delete Transactions
- Income & Expense Tracking
- Interactive Charts (Chart.js)
- Monthly Budget System
- PDF Report Generation
- Responsive Bootstrap UI
- SQLite Database
- Flask-WTF Form Validation

## Technologies Used

- Python
- Flask
- SQLite
- SQLAlchemy
- Bootstrap 5
- Chart.js
- HTML/CSS/JavaScript

## Run Project

```bash
python run.py
```
