# Expense Tracker

A modern Flask-based Expense Tracker web application with:

- User Authentication
- Add/Edit/Delete Transactions
- Income & Expense Tracking
- Interactive Charts (Chart.js)
- Monthly Budget System
- PDF Report Generation
- Responsive Bootstrap UI
- SQLite Database
- Flask-WTF Form Validation
- AI Financial Assistant: Integrated AI chatbot (powered by Groq API) that understands natural language to assist with expense queries and budgeting advice.
- Intelligent Tracking: Add, Edit, and Delete transactions with support for Categories and Sub-categories.
- User Authentication: Secure login/signup system with session management.
- Visual Analytics: Interactive charts (Chart.js) to visualize spending habits.
- Reporting: Automatic PDF Report generation for monthly summaries.
- Responsive Design: Fully mobile-friendly UI built with Bootstrap 5.

## Technologies Used

- Python
- Flask
- SQLite
- SQLAlchemy
- Bootstrap 5
- Chart.js
- HTML/CSS/JavaScript

## Run Project

```bash
python run.py
```
